### IPFS Integration for Lock Commands

**Feature**: Added IPFS sync capability to lock management commands.

**What's New**:
- `mcli lock write` now supports `--to-ipfs` flag for automatic IPFS push after writing
- `mcli lock update` now supports `--to-ipfs` flag for automatic IPFS push after updating
- Both commands support `--description/-d` flag for adding custom descriptions to IPFS syncs

**Usage Examples**:
```bash
# Write lockfile and push to IPFS
mcli lock write --to-ipfs -d "My lock state v1.0"

# Update lockfile and push to IPFS
mcli lock update --to-ipfs -d "Updated production lockfile"
mcli lock update -g --to-ipfs -d "Global lockfile update"
```

**Impact**: Users can now automatically create immutable backups of their lockfiles on IPFS when writing or updating them, ensuring version tracking and decentralized backup.
