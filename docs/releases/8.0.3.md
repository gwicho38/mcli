# Release Notes - Version 8.0.3

**Release Date:** 2025-12-28

## Overview

Version 8.0.3 is a patch release that adds native script support to the edit command and fixes several CI/CD pipeline issues.

## New Features

### Native Script Support in Edit Command

**Description:** The `mcli edit` command now supports editing native Python and shell scripts directly, not just legacy JSON workflow files.

```bash
# Edit native Python script
mcli edit my-python-workflow

# Edit native shell script
mcli edit my-shell-script

# Still supports legacy JSON files
mcli edit legacy-json-workflow
```

**Technical Details:**
- Edit command searches for `.py`, `.sh`, and `.json` files in order of preference
- Uses the `EditMessages` constants class for consistent messaging
- Falls back gracefully when scripts aren't found

## Bug Fixes

### CI/CD Pipeline Fixes

1. **Bandit Security Scan Configuration**
   - Added skip list for CLI tool false positives (B101, B102, B104, B108, B113, B301, B306, B310, B324, B404, B602, B603, B604, B605, B607)
   - Set severity filter to HIGH to reduce noise
   - Applied consistent configuration across `security.yml` and `ci.yml`

2. **Python Version Matrix Update**
   - Removed Python 3.9 from test matrix (project requires >= 3.10)
   - Updated `ci.yml` and `test.yml` to use Python 3.10, 3.11, 3.12

3. **Pre-commit Hook Cleanup**
   - Removed `safety` pre-commit hook (only supported Poetry, not UV)
   - Security checks now run exclusively in CI via `security.yml`

4. **Code Formatting Fixes**
   - Fixed black formatting issues in 4 files
   - Fixed isort import ordering issues
