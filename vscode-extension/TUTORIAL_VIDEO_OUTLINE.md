## Visual Effects & Editing Notes

### Callouts to Add
- Arrow pointing to "Run" button
- Highlight "Shift+Enter" keyboard shortcut on screen
- Zoom in on cell output when it appears
- Circle the "+ Add Cell" button when mentioned

### Transitions
- Smooth fade between scenes
- Quick cuts for action steps
- Slow-mo on cell execution (output appearing)

### Background Music
- Upbeat, modern tech music
- Volume: Low, doesn't overpower voice
- Suggested: Royalty-free "tech tutorial" music

### Text Overlays
- Bold, easy-to-read font (San Francisco or Roboto)
- Color: Match extension branding (purple/orange)
- Animation: Fade in, stay, fade out (not distracting)

## Recording Checklist

### Before Recording
- [ ] Clean VSCode workspace (no other extensions visible)
- [ ] Example files ready in workspace
- [ ] VSCode theme: Dark+ (default dark)
- [ ] Font size: Increase to 16-18pt for readability
- [ ] Zoom level: 150-200% for screen recording
- [ ] Hide menu bar if possible
- [ ] Close unused panels (terminal, problems, etc.)
- [ ] Cursor highlighting enabled (for better visibility)

### During Recording
- [ ] Record in 1080p minimum (4K preferred)
- [ ] 30 fps or higher
- [ ] Clear audio (use good microphone)
- [ ] Speak slowly and clearly
- [ ] Pause between sections for editing
- [ ] Re-record if you make a mistake
- [ ] Show mouse movements clearly

### After Recording
- [ ] Add intro/outro graphics
- [ ] Add background music
- [ ] Add text overlays and callouts
- [ ] Color grade for consistency
- [ ] Add captions/subtitles
- [ ] Export in multiple resolutions (1080p, 720p)
- [ ] Test on different devices
